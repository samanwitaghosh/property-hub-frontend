# property-hub

A React application to display multiple real estate properties with images, videos, and embedded maps.  
All property data (text, images, videos) is stored in a single platform using GitHub JSON files.

## Features

- Display multiple properties at once
- Each property includes:
  - Name and address
  - Embedded Google Map based on location
  - Images gallery
  - Videos
- Fully frontend-based: no backend required
- Free and fully hosted using GitHub + Vercel/Netlify

## Tech Stack

- **Frontend:** React.js
- **Data Storage:** GitHub JSON files
- **Media Hosting:** GitHub-hosted images/videos
- **Map Integration:** Google Maps Embed
